  <action>
Fully replace database/connection.py. The file must implement:

1. Module-level pool variable:
   ```python
   _pool: asyncpg.Pool | None = None
   ```

2. Pool size constants read from environment (DB-12 + roadmap env-var requirement):
   ```python
   import os
   _POOL_MIN = int(os.getenv("DB_POOL_MIN", "5"))
   _POOL_MAX = int(os.getenv("DB_POOL_MAX", "20"))
   _CMD_TIMEOUT = 30
   ```

3. async def init_pool(dsn: str) -> None:
   - Uses asyncpg.create_pool() with min_size, max_size, command_timeout
   - Registers a JSONB codec on each connection via init= callback:
     ```python
     import json
     async def _init_connection(conn):
         await conn.set_type_codec(
             'jsonb', encoder=json.dumps, decoder=json.loads, schema='pg_catalog'
         )
     ```
   - Logs pool initialised with min/max/timeout values

4. async def close_pool() -> None:
   - Awaits _pool.close() if _pool is not None
   - Logs pool closed

5. @contextlib.asynccontextmanager get_db_conn():
   - Asserts _pool is not None (error message: "Pool not initialised — call init_pool() first")
   - async with _pool.acquire() as conn: yield conn
   - Return type annotation: AsyncGenerator[asyncpg.Connection, None]

6. Remove ALL psycopg2 imports. Remove threading.Lock, ThreadedConnectionPool,
   get_sync_cursor, get_sync_connection, init_schema.

In requirements.txt:
- Remove the line: psycopg2-binary>=2.9,<3.0
- Add under the Database section: asyncpg>=0.29,<1.0
